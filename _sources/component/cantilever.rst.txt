Cantilever
==========

Summary
-------

An object to store cantilever parameters, needed to convert a spring
constant shift to a resonance-frequency shift. 

Example Usage
^^^^^^^^^^^^^

Create a cantilever object with spring constant k 7.8e5 aN/nm and
frequency of 4.975e6 mHz::

    cantilever = Cantilever(k_c=7.8e5, f_c=4.975e6)

Print out the cantilever summery::

    print(cantilever)
